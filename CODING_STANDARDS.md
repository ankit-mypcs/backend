# MYPCS.IN Coding Standards
- Every file: docstring with PURPOSE, USED BY, DEPENDS ON
- Every function: one-line docstring
- Complex logic: WHY comments not WHAT
- Models: PascalCase singular. Views: verb+noun. URLs: lowercase hyphens.
- No file > 300 lines. No function > 40 lines.
- Git commits: feat/fix/docs/refactor prefix
- See full standards in project docs.
